# AI Loan Assistant Chatbot 🤖

An AI-powered chatbot that assists users with **loan-related queries**, eligibility information, and general financial guidance.
The chatbot uses **Natural Language Processing (NLP) concepts and machine learning techniques** to understand user input and respond in a conversational format.

This project demonstrates how **AI-based virtual assistants can automate customer support in financial services**, helping users quickly access loan-related information.

---

## Features 🚀

* Interactive chatbot for loan-related queries
* Basic NLP-based intent recognition
* Quick responses to common financial questions
* Modular and easy-to-extend project structure
* Designed as a prototype for financial assistance chatbots

## Installation and Setup ⚙

### 1. Clone the repository

```bash
git clone https://github.com/Shivani7122/AI-Loan-Assisstant-Chatbot.git
```

### 2. Navigate to the project folder

```bash
cd AI-Loan-Assisstant-Chatbot
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```
